# KOREA Book Cover Dataset
Korea Book Cover Dataset

한국 책표지 데이터셋입니다.


## Detail
총 6개 카테고리의 약 50,000장 가량의 책 표지 데이터가 있습니다.

각 카테고리는
- 예술
- 유아
- 소설 및 시
- 종교
- 과학
- 여행

이며 jpg형식의 파일로 되어 있습니다.
